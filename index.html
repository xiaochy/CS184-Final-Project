Final Project Proposal

A Reconstruction of Snow Simulation

Claire Fang, Chuyang Xiao, Yifan Yin, Xiaoru Zhao

Summary

We plan to reconstruct the MPM method to simulate the snow behaviors, discussed in A material point method for snow simulation. This includes building a grid, computing particle volumes and velocities, handling collisions, etc. If we have extra time, we also plan to explore other simulation methods to improve the effect, handle the interaction with air, which is ignored in the original paper, and leverage ML methods to improve the speeding and accuracy of estimating snow-related values in the model.

Problem Description

Snow dynamics is difficult to simulate as it has both solid- and fluid-like properties. But it is really important to simulate snow for its wide applications such as in movies and animations. Since it’s not only expensive to build a real scene with snow but also dangerous for actors to act in such a scene. We will be building off of existing method utilizing Material Point Method with a elastoplastic constitutive model to describe different snow behavior.

Goals and Deliverables

  • We will reconstruct A material point method for snow simulation as a baseline method to simulate snow. (including different types of snow & different behaviors of snow: self-collisions, collisions with other objects (like air) or collisions with snow of different types)
  • We plan to make snow simulation videos implementing the collision effect mentioned above. We will focus on simulating certain snow behaviors based on the given imput parameters. (If we have extra time, we may explore how to make it into an interactive system so that users can change the parameters and simulate their own snow!)
  • We plan to measure our system by the closeness of our simulation’s resemblance to snow. We can create different scenarios like collisions with expectations of how it would perform.
  • We can measure the basic quality by comparing our rendering results with the graphics shown in the paper. Since this simulation has been reconstructed by many people in recent years, we can also value our performance by referring to other interesting snow simulation images using MPM method.
  • If we are ahead of plan, we hope to do the following.
    • We may explore other simulation methods to improve the simulation effect.
    • We may change data structures mentioned in the paper (using adaptive sparse grids) to speed up snow simulation.
    • We may leverage machine learning method to accelerate the process of predicting particles’ position, velocity predictions.
    • We compare the rendering images using above changes with the images using the original implementation to find the pros and cons of different methods.

Questions we hope to answer:

  • We will compare different simulation methods (MPM and so on) in order to find the current best method to simulate snow (try to solve the limitation of MPM!)
  • We will analyze the simulation time before and after we change the grid data structure to find out whether we have accelerated the simulation.
